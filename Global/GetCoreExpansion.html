Method Global:GetCoreExpansion [-] [+]

Returns emulator expansion. Expansion is 0 for classic, 1 for TBC, 2 for WOTLK and 3 for cataclysm

Synopsis

version = Global:GetCoreExpansion()

Arguments

None.

Returns

number version

Emulator expansion ID.

Valid numbers: integers from -2,147,483,647 to 2,147,483,647.
